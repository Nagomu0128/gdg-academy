# やってみよう

`script.ts` の `add` と `repeat` に型をつけて完成させます。

```ts
function add(a: number, b: number): number {
  return a + b;
}

function repeat(text: string, times: number): string {
  return text.repeat(times);
}
```

手順

1. `add` の引数・戻り値に `: number` をつけ、`return a + b;` を書く
2. `repeat` の引数・戻り値に型をつけ、`text.repeat(times)` を `return` する
3. 「▶ 実行」で `7` と `ababab` が出るのを確認して「できた!」を押す
