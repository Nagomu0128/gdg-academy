# やってみよう

`script.ts` の `label` に型をつけ、`typeof` で分岐させます。

```ts
function label(value: string | number): string {
  if (typeof value === "string") {
    return "文字列: " + value;
  }
  return "数値: " + value;
}
```

手順

1. 引数 `value` に `: string | number`、戻り値に `: string` をつける
2. `typeof value === "string"` で分岐し、それぞれの文字列を `return` する
3. 「▶ 実行」で `文字列: こんにちは` と `数値: 42` が出るのを確認する
