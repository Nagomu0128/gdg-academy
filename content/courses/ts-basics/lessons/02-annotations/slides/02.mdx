# 変数につける型注釈

変数名のうしろに `: 型名` を書きます。

```ts
let userName: string = "みらい";
let age: number = 20;
let isMember: boolean = true;
```

- 文字列は `string`、数値は `number`
- true / false は `boolean`(真偽値)
- `=` より前、変数名の直後に書きます

<Callout type="warn">
`:` の前は名前、うしろは型名です。値そのもの(`"みらい"`)ではなく、型の名前(`string`)を書きます。
</Callout>
