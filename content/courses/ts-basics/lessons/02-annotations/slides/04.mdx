# やってみよう

`script.ts` の 3 つの変数と、関数 `summary` に型注釈をつけます。

```ts
// 変数(= の前に型を書く)
let price: number = 500;

// 関数(引数と ) のうしろに型を書く)
function summary(name: string, yen: number): string {
  return name + " は " + yen + " 円です";
}
```

手順

1. `productName` `price` `inStock` に `: string` `: number` `: boolean` をつける
2. `summary` の引数と戻り値に型注釈をつけ、`return` の 1 行を書く
3. 「▶ 実行」で結果を確認して「できた!」を押す
