# やってみよう

エディタの `script.ts` にある 2 つの変数に型注釈を付けます。

```ts
// 書き換える前
let libraryName = "TypeScript";
let version = 5;

// 書き換えた後の形
let libraryName: string = "TypeScript";
let version: number = 5;
```

手順

1. `libraryName` に `: string` を付ける
2. `version` に `: number` を付ける
3. 「▶ 実行」で `TypeScript v5` が出るのを確認して「できた!」を押す

<Callout type="info">
`describe` 関数と `console.log` の行はそのままで大丈夫です。
</Callout>
