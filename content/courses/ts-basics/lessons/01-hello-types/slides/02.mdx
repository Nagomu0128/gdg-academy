# 型注釈とは

変数に「この変数には文字列しか入れない」という約束を書き添えるのが型注釈です。

```ts
let userName: string = "みらい";
let age: number = 20;
```

- 変数名のうしろに `: 型名` を書きます
- 文字列は `string`、数値は `number`、真偽値は `boolean`
- 約束と違う値を入れようとすると、実行する前にエディタが教えてくれます

```ts
let age: number = 20;
age = "はたち"; // エラー: number に文字列は入れられない
```

<Callout type="warn">
`:` の前は変数名、うしろは型名です。`let age: number = 20;` のように `=` より前に書きます。
</Callout>
