# 最初のコミットまでの 4 コマンド

```
git init
echo "# メモ帳アプリ" > README.md
git add README.md
git commit -m "最初のコミット"
```

1. `git init` — このフォルダをリポジトリにする
2. `echo "内容" > ファイル名` — ファイルをつくる(ターミナルの書き方)
3. `git add README.md` — README.md をステージに載せる
4. `git commit -m "メッセージ"` — ステージの内容を記録する

<Callout type="warn">
コミットには必ず `-m "メッセージ"` を付けます。メッセージは半角の `"` で囲みましょう。
</Callout>
