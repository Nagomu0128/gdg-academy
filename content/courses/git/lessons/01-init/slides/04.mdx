# やってみよう

`commands.sh` に、スライドの 4 コマンドを上から順に書きます。

- `#` で始まる行はコメント(メモ)です。そのまま残して、各コメントの下にコマンドを書きましょう
- 書くたびに右のターミナルで実行結果が再生されます
- `git commit` のあとに `git status` を足すと「きれいな状態」も確認できます(任意)

うまくいくと、ターミナルの最後に

```
[main xxxxxxx] 最初のコミット
```

のような表示と、履歴のグラフが出ます。確認できたら「できた!」を押しましょう。

<Callout type="info">
赤い行はエラーです。コマンドの綴りや `"` の閉じ忘れを見直しましょう。
</Callout>
