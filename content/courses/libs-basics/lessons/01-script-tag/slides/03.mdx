# dayjs で日付を整形する

dayjs は「日付の読み書き」を簡単にするライブラリです。

```js
const day = dayjs("2026-01-01");
const text = day.format("YYYY年MM月DD日");
// → "2026年01月01日"
```

- `dayjs("2026-01-01")` で日付オブジェクトを作ります
- `format` に書式(`YYYY` = 年 4 桁、`MM` = 月 2 桁、`DD` = 日 2 桁)を渡すと文字列になります
- 書式の中の日本語(年・月・日)はそのまま出力されます

<Callout type="info">
このレッスンでは判定を安定させるため、現在時刻ではなく固定の日付 "2026-01-01" を使います。
</Callout>
